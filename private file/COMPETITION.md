# AI활용 차세대 교육 솔루션 공모전

---

## 공모전 주제
**AI활용 차세대 교육 솔루션**
- 교육 현장의 페인 포인트를 AI 기술로 해결하는 솔루션 기획 및 구현
- 단순 LMS 구축이 아닌, 교강사/수강생/운영자의 실질적 문제를 해결하는 AI솔루션

---

## 제출 기한
**2026.04.13(월)**
- 기한 이후 접수 → 심사 제외
- 기한 이후 Commit → 부정행위로 탈락 처리 가능

---

## 최종 제출물
1. **GitHub 저장소 주소** (public 설정, API Key 노출 금지)
2. **배포된 라이브 URL**
3. **AI 리포트** (지정 양식, PDF)
4. **개인정보 수집/이용 동의서 + 참가 각서** (서명 완료 PDF)

---

## 심사 기준

| 항목 | 내용 |
|------|------|
| 기술적 완성도 | 구현 수준, 안정성, 배포 상태 |
| AI활용 능력 및 효율성 | AI를 얼마나 효과적으로 활용했는가 |
| 기획력 및 실무 접합성 | 실제 교육 현장 문제 해결 가능성 |
| 창의성 | 기존 솔루션과의 차별점 |

> **AI와 효율적으로 협업하는 능력**을 중요 심사 항목으로 평가.
> AI와 기획한 기획문서, 지침서 등을 프로젝트 내부에 포함 권장.
