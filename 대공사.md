# 대공사 — localStorage → Firestore 전수 이전

> 목적: 브라우저 공유 localStorage로 인한 계정 간 데이터 누수 완전 제거 + 실시간 동기화  
> 결정: **B (전수 Firestore 이전)** + **D 세션 단순화 (useRef, 멀티탭 킥 허용)**  
> 진행: Phase 0 → 4 → 1 → 2 → 3 → 5 순서

---

## ✅ 확정된 결정사항

| 항목 | 결정 |
|---|---|
| 저장 방식 | 전수 Firestore 이전 (B) |
| 세션 방식 | useRef + 로그인 시 sessionId 재생성. 멀티탭 킥 허용 (D 단순화) |
| 디버그 패널 권한 | 관리자 전용 (⚠️ Dev Only 표시) |
| 디버그 편집 범위 | 모든 학생 선택 후 편집 가능 |
| questRepeat 저장 | Firestore 서브컬렉션 `users/{uid}/questHistory/{date}` |
| 기존 데이터 보존 | Firestore를 source of truth로 사용, localStorage 마이그레이션 불필요 |
| questProgress (진행 임시상태) | sessionStorage 유지 (빈번한 쓰기, 서버 불필요) |
