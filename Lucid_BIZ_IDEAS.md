# Lucid — 비즈니스 모델 아이디어

> 공모전 이후 또는 실제 도입 검토용 아이디어 메모입니다.

---

## 아이디어 1. 멘토/학원 수수료 모델

국비 부트캠프 특성상 학원과 멘토는 학생이 유지될수록 수수료를 받는 구조 (보험사-보험 설계사 관계와 동일).

Lucid가 학생 출석/활동 데이터를 보유하므로, 학생이 플랫폼에서 활동한 일수 기준으로 멘토/학원에 수수료를 정산하는 구조를 만들 수 있다.

- 학원 입장: 이탈률 감소 + 수강생 관리 데이터 확보
- 멘토 입장: 학생 유지가 곧 수익 → 방치 불가 구조
- 부가 효과: 쿠폰 지급 타이밍에 멘토가 학생에게 자연스럽게 연락하는 접점 생성

---

## 아이디어 2. 따즈아 VOD 연동 (인프런 유사 플랫폼)

따즈아 계열의 단품 영상 강의 플랫폼과 연동.

학생이 루틴이나 문제지옥에서 막히는 개념을 AI가 감지 → 관련 VOD 자동 추천 → Lucid 안에서 바로 결제/시청.

- 단품: 300원
- 구독: 넷플릭스 방식 월정액
- 온보딩 프로모션: 학원이 신규 학생에게 1,500원 선충전 → VOD 5개 맛보기 제공 (첫 결제 허들 제거, 비용은 학원 부담, 전환율은 플랫폼이 가져감)

맥락 기반 추천이므로 광고가 아니라 도움으로 인식됨. 외부 유튜브/GPT로 이탈하는 트리거를 내부에서 해결하는 구조.

---

## 아이디어 3. XP → 재화 → 쿠폰 환원 루프

학생이 쌓은 XP가 실물 보상으로 연결되면 플랫폼 이탈 동기가 줄어듦.

- 루틴 성실 이수 14일 → 2,000원 커피 쿠폰
- 루틴 성실 이수 28일 → 4,000원 아이스아메리카노 쿠폰
- 쿠폰 재원: 멘토/학원 수수료의 일부 환원

학생 입장에서 공부하면 돈이 되고, 학원 입장에서는 이탈 방지 비용이 마케팅 비용보다 저렴한 구조.

---

## 선순환 요약

```
학생 열심히 함
→ 출석 유지
→ 멘토 수수료 발생
→ 수수료 일부 쿠폰으로 환원
→ 학생 더 열심히 함
→ VOD 막힌 개념 해결
→ 따즈아 수익 발생
→ 플랫폼 전체 유지
```
