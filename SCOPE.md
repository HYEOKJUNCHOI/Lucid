## 📬 이메일 메타포 (파일 상태 3종)

```
📬 NEW     — GitHub commit 날짜 = 오늘 + 아직 안 열어봄
📩 UNREAD  — 올라온지 됐지만 아직 안 열어봄 (NEW 아님)
📭 READ    — 코드 보고 퀴즈까지 완료 → 보관함 (원할 때 복습)
```

### 퀘스트 배정 우선순위
```
NEW 있음  → 퀘스트에 NEW 먼저
NEW 없음  → UNREAD 배정
둘 다 없음 → READ 중 복습 배정
```

### 파일 1개 완료 판단
```
코드 열람 → AI 설명 or 스킵 → 퀴즈 3문제 완료
                                    ↓
                              ✅ READ 마킹 (localStorage)
```

### 퀘스트 전체 완료 판단
```
배정된 미션 전부 READ → 전체 완료 → 🫘 23% 드랍 기회
일부만 READ         → 부분 완료 → XP만 (원두 없음)
```

### localStorage 구조 변경
```js
// 기존
lucid_visited_files = ["ch01/Main.java", ...]   // 단순 방문 목록

// 신규
lucid_file_status = {
  "ch01/Main.java": "read",     // 퀴즈까지 완료
  "ch02/Dog.java": "visited",   // 열어봤지만 퀴즈 미완료
}
```

**NEW 판단:** GitHub API로 각 파일의 마지막 commit 날짜 fetch → 오늘 날짜와 비교.

## 🫘 원두 드랍 로직

```
첫 퀘스트 완료:
  → 🫘 확정 드랍 1개 (튜토리얼, BeanRewardCard 표시)

이후 퀘스트 완료 (전체 클리어 시에만):
  → Math.random() < 0.23 → 🫘 1개 드랍
  → 부분 완료는 XP만

교환:
  - 원두 5개 → ☕ 커피 1잔 (교환 UI는 v2)
  - 평균 주기: 약 22일 (운 좋으면 10일, 운 나쁘면 35일+)
  - 기수 평균 5잔, 한 달 1잔 체감
```

### 첫 원두 획득 카드 문구 (확정)
```
☕ 첫 원두를 얻었어요

퀘스트를 완료하면 확률적으로 원두가 드랍됩니다.

원두 5개를 모으면 커피 한 잔으로 교환할 수 있어요.
```

### 빛나는 효과 (4겹)
1. **배경 아우라** — 카드 뒤 500px 블러 원 (앰버 계열)
2. **카드 테두리 글로우** — box-shadow 이중 글로우
3. **원두 이모지 빛남** — text-shadow + 느린 펄스 애니메이션 (2초 주기)
4. **샤인 스윕** — 카드 위를 1회 지나가는 사선 하이라이트

### 플레이버 문구 (B 확정)
> "깊은 향이 감도는 원두 한 알. 부지런한 자에게만 허락된 보상이다."

## 🚫 v2 — 공모전 제출 후

### 시스템급 (큰 작업)
- **LP 시스템** — 과목별 티어 (아이언~다이아), 문제지옥 전용
- **문제지옥 모드** — 과목 선택, 배치고사, 무한 문제, 실시간 LP 증감
- **시즌 시스템** — 2개월 리셋, LP 0 + 티어 재배치
  - 기수(5개월) = 시즌 2.5번
  - 시즌 종료 → LP 0 리셋, 티어 재배치 (-1단계 or 재배치고사)
  - 기수 종료 → 전체 데이터 폐기
- **온보딩** — "어떤 걸 배우고 싶나요?" + 멀티 선택, 단일 추천
- **커피 교환소** — 원두 5개 → 커피 1잔 교환 UI
- **배치고사** — 10문제 풀고 티어 결정

### 퀘스트 확장
- **일일퀘스트 4미션 구성** — 챕터/문제지옥/약점보강/자유학습
  - 현재는 기존 루틴 로직(파일 기반) 유지
- **약점 보강 미션** — 기존 `lucid_weak_files` 데이터 활용
- **자유학습 모드** — AI 주제 자동 생성

### 기능 고도화
- **메타포 v2** — 피드백 기반 프롬프트 자동 개선, 학생 유형별 개인화
- **강사 대시보드 두 축 교차** — streak × 티어 → 학생 상태 판별 (면담/동기부여/위험)
- **3개월 마일스톤** — 강사 면담 기준 알림
- **NEW 뱃지 실시간 표시** — 강사 push 감지 (v1에서 기본만 구현)

### 파서 레이어 (진짜 범용화)
- **PDF 파싱** — pdf-parse
- **Word 파싱** — mammoth.js
- **Excel 파싱** — sheetjs
- **HWP/HWPX 파싱** — hwp.js
- 이게 구현되면 "어떤 교육 자료든 AI가 이해시켜주는 엔진" 완성

## 🎨 XP vs LP 구분 (결정)

| 모드 | XP (→ 커피·원두) | LP (실력 티어) |
|------|:---:|:---:|
| 챕터별 학습 | ✅ | ❌ (오픈북이라 측정 불가) |
| 퀘스트 | ✅ | ✅ (적게) — v2 |
| 문제지옥 | ✅ | ✅ (많이) — v2 |
| 자유학습 | ✅ | ❌ — v2 |

- **퀘스트와 문제지옥은 LP 공유** (Java 선택 → 퀘스트 LP가 문제지옥 Java 티어에 반영)
- **전체 LP 없음** — 과목별 LP만 존재
- **XP는 어디서 쌓든 = 커피/원두 교환** (성실성 보상)
- **LP는 맨몸 실력 측정 모드에서만** (실력 증명)
