## 1. 강의실 레이아웃

> 실제 강의장 구조 기반. 강사/멘토 모두 동일 화면 사용.

```
[              칠판              ]
                                    [   강사   ]
[학생][학생][학생][학생]  [학생][학생][학생][학생]
[학생][학생][학생][학생]  [학생][학생][학생][학생]
[학생][학생][학생][학생]  [학생][학생][학생][학생]
[학생][학생][학생][학생]  [학생][학생][학생][학생]
[학생][학생][학생][학생]  [학생][학생][학생][학생]
```

- 4×2 블록 × 5줄 = 40자리 기본
- 실제 수강생 수에 맞게 빈 자리는 비워둠
- 드래그로 학생 자리 이동 가능

## 2. 학생 카드 (피파 카드 스타일)

### 카드에 표시되는 정보

| 항목 | 설명 |
|------|------|
| 이름 | 학생 이름 |
| 레벨 | 루시드 전체 레벨 (Lv.N) |
| 오늘 접속 여부 | O / X |
| 연속 출석 일수 | N일 연속 |
| 이번 주 루틴 클리어율 | N% |
| 상태 신호 | 🟢 🟡 🔴 |

### 중도 포기자

- 카드에 ✕ 표시 (배틀로얄 탈락 느낌)
- 흐리게 처리

---

## 3. 이탈 신호 기준

> 우선순위: 출석 > 점수(활동) > 루틴

| 신호 | 조건 |
|------|------|
| 🟢 초록 (정상) | 출석 O + 활동 있음 (XP 발생) |
| 🟡 노랑 (주의) | 출석은 했으나 활동 없음 OR 2일 미접속 |
| 🔴 빨강 (위험) | 3일 이상 미접속 OR 접속했으나 XP 0 (아무것도 안 함) |

- 카드 테두리 색으로 상태 표시
- 🔴 학생은 대시보드 상단에 자동 노출 (눈에 띄게)

## 4. AI 밸런스 패치 (4인 1조 자동 편성)

- 버튼 하나로 AI가 자동으로 4인 1조 팀 편성
- 편성 기준: 레벨 + 적극도(루틴 클리어율) 혼합

```
고레벨 1명 + 중레벨 2명 + 저레벨 1명 → 멘토링 효과
```

- 편성 결과는 강의실 레이아웃에 자리로 반영
- 멘토/강사가 수동으로 조정 가능 (드래그)

---

## 5. PDF 출력

- 수업 중 강사용 좌석표 출력
- 출력 내용: **이름 + 자리 위치만** (레벨/점수 제외)
- 레이아웃은 강의실 구조 그대로 출력

```
[              칠판              ]
                            [강사]

[김철수][이영희][박민준][최수아]  [정하늘][윤서준][강다은][임태양]
...
```

## 6. Firestore 데이터 구조 (참고)

```
user/{uid}
  - name: string
  - level: number
  - todayAttended: boolean       // 오늘 접속 여부 (자정 리셋)
  - streak: number               // 연속 출석 일수
  - lastStudiedAt: timestamp     // 마지막 접속 시각
  - dailyXP: number              // 오늘 획득 XP
  - weeklyRoutineClear: number   // 이번 주 루틴 클리어율 0~100%
  - status: "green" | "yellow" | "red" | "dropout"  // 이탈 신호
  - seatPosition: number         // 강의실 자리 번호 (1~40)
```
