# Lucid v1.0 기능 정리

## 1. 학생 메인 화면 (StudentPage)
- **역할**: 홈 대시보드
- **주요 기능**:
  - 사용자 상태 표시 (이름, 레벨, XP, 스트릭)
  - 일일 퀘스트 완료 상태
  - 주간 캘린더 (출석/결석/얼리기 표시)
  - 월간 XP 및 원두 현황
  - 역할별 메뉴 (학생/강사/관리자)
- **AI 모델**: Gemini (응원멘트 5종 로테이션)

---

## 2. 자유학습 (FreeStudyView) ⭐ 핵심
- **역할**: 범용 학습 엔진, 복습&예습
- **주요 기능**:
  - GitHub 강사 코드 자동 불러오기 (기수 필터링)
  - Monaco 에디터로 코드 열람
  - AI 튜터 채팅 (코드 컨텍스트 기반 질문)
  - AI 키워드 코드생성 (예시 코드 자동 작성)
  - AI 문제출제 (자동 퀴즈 생성 및 풀이)
  - 용어 사전 (더블클릭 팝업)
  - 코드노트 (마크다운)
  - 메모장 (마크다운 + 저장/PDF)
  - 타이핑 연습
- **AI 모델**:
  - `gemini-2.5-flash-lite` — 튜터 채팅, 코드생성, 퀴즈생성
  - `gemini-2.0-flash` — 타자 번역 (한→영)

---

## 3. 문제풀기 (LevelUpView)
- **역할**: 실력 측정 및 티어 경쟁
- **주요 기능**:
  - 과목별 입장 (Java 기초, Python AI 등)
  - 배치고사 (초기 레벨 진단 - v2 예정)
  - 일반 퀴즈 (XP + LP 획득)
  - 현재 티어/LP 표시
  - 주간 우수자 랭킹
  - 하트 시스템 (시각적 피드백)
  - 스트릭 얼리기 UI
- **AI 모델**: Gemini (퀴즈 생성)

---

## 4. 관리자 페이지 (AdminPage)
### 4-1. 학생관리 (StudentManagement)
- 신규 학생 정보 등록 (이메일, 이름, 기수, 난이도)
- 기수별/그룹별 학생 조회
- 학생 정보 수정
- 일괄 그룹 배정
- 학생 삭제

### 4-2. 학생 현황 대시보드 (StudentDashboard)
- 전체 학생 카드형 리스트
- 레벨, 스트릭, 활동 상태 표시
- 정렬 (ㄱㄴㄷ순, 레벨순)
- 학생 상세 조회

### 4-3. 자리배치 (SeatChart)
- 모의 테이블 레이아웃
- AI 추천 배치 (Level 분산)
- 수동 배치 조정
- 테이블별 편성
- **AI 모델**: Gemini (배치 최적화)

### 4-4. 디버그 패널 (DebugPanel)
- 학생 선택 후 Firestore 필드 직접 편집
- streak, XP, 레벨, 원두, 아메리카노 등 실시간 수정
- 타자 기록 편집
- 전체 학생 랜덤 테스트 데이터 시드 버튼
- 출석 달력 조회

---

## 5. 기타 기능

### 5-1. 용어 사전 (DictionaryPopup)
- **작동**: 텍스트 더블클릭 → 팝업
- **기능**: 용어 번역/설명
- **AI 모델**: `gpt-4.1-nano` → **Gemini로 통일 필요**

### 5-2. 타이핑 연습 (TypingPractice)
- 한글 코드 → 영문 자동 변환
- 타수 기록
- 정확도 추적

### 5-3. 메모패널 (MemoPanel)
- 마크다운 에디터
- 저장/로드
- PDF 내보내기

---

## 6. 백그라운드 서비스

### 6-1. userStateService
- 스트릭 계산 및 업데이트
- 얼리기 로직
- 출석 캘린더 생성

### 6-2. learningService
- 퀴즈 생성 (현재 dead code: generateFreeStudyCode)
- 타자 통계 기록
- XP 계산

### 6-3. dictionaryService
- 용어 캐싱

## 8. 데이터 흐름

```
학생 입장
  ├─ GitHub 코드 불러오기 (캐시: SHA)
  ├─ AI 튜터 채팅 (코드 컨텍스트)
  ├─ 용어 사전 (더블클릭)
  ├─ 퀴즈 풀기
  └─ XP/스트릭 업데이트 (Firestore)

관리자 입장
  ├─ 학생 정보 관리
  ├─ 자리배치 AI 추천
  ├─ 학생 현황 대시보드
  └─ 디버그 패널 (직접 편집)
```
